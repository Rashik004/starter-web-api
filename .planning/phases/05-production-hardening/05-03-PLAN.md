2. Create `src/Starter.HealthChecks/Options/HealthCheckModuleOptions.cs`:
   - `namespace Starter.HealthChecks.Options;`
   - `public sealed class HealthCheckModuleOptions` with `public const string SectionName = "HealthChecks";`
   - Properties:
     - `public string ExternalServiceUri { get; set; } = "";` -- URI for the sample external dependency health check. Empty string = check is registered but reports Degraded with "No URI configured" message.
     - `public int TimeoutSeconds { get; set; } = 5;` -- HTTP timeout for external service probe

3. Create `src/Starter.HealthChecks/Checks/ExternalServiceHealthCheck.cs`:
   - `namespace Starter.HealthChecks.Checks;`
   - `internal sealed class ExternalServiceHealthCheck : IHealthCheck` using primary constructor:
     `internal sealed class ExternalServiceHealthCheck(IHttpClientFactory httpClientFactory, IOptions<HealthCheckModuleOptions> options) : IHealthCheck`
   - `CheckHealthAsync` implementation:
     - If `options.Value.ExternalServiceUri` is null or whitespace, return `HealthCheckResult.Degraded("External service URI not configured. Set HealthChecks:ExternalServiceUri in appsettings.json.")`
     - Otherwise, create an HttpClient via `httpClientFactory.CreateClient()`, set `Timeout = TimeSpan.FromSeconds(options.Value.TimeoutSeconds)`
     - Try: `var response = await client.GetAsync(options.Value.ExternalServiceUri, cancellationToken);`
     - If `response.IsSuccessStatusCode`: return `HealthCheckResult.Healthy($"External service is reachable ({response.StatusCode}).")`
     - Else: return `HealthCheckResult.Degraded($"External service returned {(int)response.StatusCode} {response.StatusCode}.")`
     - Catch `TaskCanceledException`: return `HealthCheckResult.Unhealthy("External service health check timed out.")`
     - Catch `HttpRequestException ex`: return `HealthCheckResult.Unhealthy("External service is unreachable.", ex)`
   - Required usings: `using Microsoft.Extensions.Diagnostics.HealthChecks;`, `using Microsoft.Extensions.Options;`, `using Starter.HealthChecks.Options;`

4. Create `src/Starter.HealthChecks/HealthChecksExtensions.cs`:
   - `namespace Starter.HealthChecks;`
   - `public static class HealthChecksExtensions`
   - `AddAppHealthChecks(this WebApplicationBuilder builder)`:
     - Register IOptions: `builder.Services.AddOptions<HealthCheckModuleOptions>().BindConfiguration(HealthCheckModuleOptions.SectionName).ValidateDataAnnotations().ValidateOnStart();`
     - Register HttpClientFactory (needed by ExternalServiceHealthCheck): `builder.Services.AddHttpClient();`
     - Register health checks:
       ```csharp
       builder.Services.AddHealthChecks()
           .AddDbContextCheck<Starter.Data.AppDbContext>(
               name: "database",
               tags: new[] { "ready" })
           .AddCheck<Checks.ExternalServiceHealthCheck>(
               name: "external-service",
               tags: new[] { "ready" });
       ```
     - Return `builder`
   - `UseAppHealthChecks(this WebApplication app)`:
     - Map aggregate endpoint:
       ```csharp
       app.MapHealthChecks("/health", new HealthCheckOptions
       {
           ResponseWriter = WriteResponse
       });
       ```
     - Map readiness endpoint (only "ready" tagged checks):
       ```csharp
       app.MapHealthChecks("/health/ready", new HealthCheckOptions
       {
           Predicate = check => check.Tags.Contains("ready"),
           ResponseWriter = WriteResponse
       });
       ```
     - Map liveness endpoint (no checks -- if process runs, it's alive):
       ```csharp
       app.MapHealthChecks("/health/live", new HealthCheckOptions
       {
           Predicate = _ => false,
           ResponseWriter = WriteResponse
       });
       ```
     - Return `app`
   - Private static `WriteResponse(HttpContext context, HealthReport report)` method:
     ```csharp
     private static Task WriteResponse(HttpContext context, HealthReport report)
     {
         context.Response.ContentType = "application/json; charset=utf-8";
         var options = new JsonWriterOptions { Indented = true };
         using var memoryStream = new MemoryStream();
         using (var jsonWriter = new Utf8JsonWriter(memoryStream, options))
         {
             jsonWriter.WriteStartObject();
             jsonWriter.WriteString("status", report.Status.ToString());
             jsonWriter.WriteString("totalDuration", report.TotalDuration.ToString());
             jsonWriter.WriteStartObject("results");
             foreach (var entry in report.Entries)
             {
                 jsonWriter.WriteStartObject(entry.Key);
                 jsonWriter.WriteString("status", entry.Value.Status.ToString());
                 jsonWriter.WriteString("description", entry.Value.Description);
                 jsonWriter.WriteString("duration", entry.Value.Duration.ToString());
                 if (entry.Value.Exception is not null)
                 {
                     jsonWriter.WriteString("exception", entry.Value.Exception.Message);
                 }
                 jsonWriter.WriteEndObject();
             }
             jsonWriter.WriteEndObject();
             jsonWriter.WriteEndObject();
         }
         return context.Response.WriteAsync(
             Encoding.UTF8.GetString(memoryStream.ToArray()));
     }
     ```
   - Required usings:
     - `using System.Text;`
     - `using System.Text.Json;`
     - `using Microsoft.AspNetCore.Builder;`
     - `using Microsoft.AspNetCore.Diagnostics.HealthChecks;`
     - `using Microsoft.AspNetCore.Http;`
     - `using Microsoft.Extensions.DependencyInjection;`
     - `using Microsoft.Extensions.Diagnostics.HealthChecks;`
     - `using Starter.HealthChecks.Options;`
