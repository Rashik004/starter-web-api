**4. AppDbContext** (`src/Starter.Data/AppDbContext.cs`):
```csharp
using Microsoft.EntityFrameworkCore;
using Starter.Data.Entities;

namespace Starter.Data;

internal class AppDbContext(DbContextOptions<AppDbContext> options) : DbContext(options)
{
    public DbSet<TodoItem> TodoItems => Set<TodoItem>();

    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        base.OnModelCreating(modelBuilder);
        modelBuilder.ApplyConfigurationsFromAssembly(typeof(AppDbContext).Assembly);
    }
}
```
Note: AppDbContext is `internal class` (not sealed) to allow Phase 4 Identity to potentially subclass or extend. The constructor uses primary constructor syntax.

**7. DataExtensions** (`src/Starter.Data/DataExtensions.cs`):
- `public static class DataExtensions`
- `AddAppData(this WebApplicationBuilder builder)` method that:
  a. Registers `DatabaseOptions` with `AddOptions<DatabaseOptions>().BindConfiguration(DatabaseOptions.SectionName).ValidateDataAnnotations().ValidateOnStart()`
  b. Reads `DatabaseOptions` from configuration via `builder.Configuration.GetSection(DatabaseOptions.SectionName).Get<DatabaseOptions>()!`
  c. Calls `builder.Services.AddDbContext<AppDbContext>((sp, options) => { ... })` with a switch expression on `dbOptions.Provider`:
     - `"Sqlite"` => `options.UseSqlite(builder.Configuration.GetConnectionString("Sqlite"), x => x.MigrationsAssembly(typeof(Starter.Data.Migrations.Sqlite.Marker).Assembly.GetName().Name))`
     - `"SqlServer"` => `options.UseSqlServer(...)` with `MigrationsAssembly(typeof(Starter.Data.Migrations.SqlServer.Marker).Assembly.GetName().Name)` and `EnableRetryOnFailure(maxRetryCount: dbOptions.MaxRetryCount, maxRetryDelay: TimeSpan.FromSeconds(30), errorNumbersToAdd: null)`
     - `"PostgreSql"` => `options.UseNpgsql(...)` with `MigrationsAssembly(typeof(Starter.Data.Migrations.PostgreSql.Marker).Assembly.GetName().Name)` and `EnableRetryOnFailure(maxRetryCount: dbOptions.MaxRetryCount, maxRetryDelay: TimeSpan.FromSeconds(30), errorCodesToAdd: null)`
     - `_` => `throw new InvalidOperationException($"Unsupported database provider: {dbOptions.Provider}. Valid values: Sqlite, SqlServer, PostgreSql")`
  d. After switch: if `dbOptions.CommandTimeout > 0` then `options.SetCommandTimeout(dbOptions.CommandTimeout)`
  e. After switch: if `dbOptions.EnableSensitiveDataLogging` then `options.EnableSensitiveDataLogging()`
  f. Registers `builder.Services.AddScoped(typeof(IRepository<>), typeof(EfRepository<>))`
  g. Registers `builder.Services.AddScoped<ITodoService, TodoService>()`
  h. Returns `builder`
- `UseAppData(this WebApplication app)` method that:
  a. Reads `DatabaseOptions` from `app.Configuration`
  b. If `options?.AutoMigrate == true`: creates a scope, gets `AppDbContext`, calls `db.Database.Migrate()`
  c. Returns `app`
